A single plugin for all major customizations in cart / checkout flow within WooCommerce.

== Description ==

Confusing product purchase process, long checkout forms, irrelevant offers - these are common reasons why people abandon carts and do not complete their orders.

Cashier for WooCommerce will change all that for good. It will optimize the entire checkout funnel in WooCommerce - cart page, checkout page, order complete page, product recommendations and many other enhancements options.

== Installation ==

1. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on 'Cashier for WooCommerce' option within WordPress admin sidebar
